## Signup and Login Forms
A simple React application with Signup and Login forms.

---

## Tech Stack
- React + TypeScript
- Tailwind CSS (Styling)
- Formik (Form Handling)
- Yup (Form Validation)
- React Router DOM (Navigation)
- Vite

---

## Getting Started
```bash
npm install
npm run dev
```

Application runs on:
```bash
http://localhost:5173/
```
